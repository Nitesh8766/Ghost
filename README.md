# Offline Ghost

A red/black/white CTF challenge for web forensics, OSINT, browser storage, and crypto.

## Host on GitHub Pages

1. Push this folder to a GitHub repository.
2. Go to repository Settings.
3. Open Pages.
4. Source: Deploy from a branch.
5. Branch: main.
6. Folder: /root.
7. Save.
8. Open the GitHub Pages URL.

## Challenge flow

Players should find:

- Page source clue
- Service worker cache name
- Old note
- OSINT image
- IndexedDB vault
- Strict browser decryptor

## Important

This version uses relative paths and is compatible with GitHub Pages project URLs like:

```text
https://username.github.io/offline-ghost/
```

Do not add organizer solutions or answer files to this repository.
